В задании требовалось смоделировать вид на черную дыру с диском с произвольного ракурса.
Так же необходимо было учитывать физические законы, в том числе искривление световых лучей(на изображениях
плоский диск обтекает черную дыру со всех сторон). 

Формат файла настроек:
ввод 11 чисел:
1. 2 первых типа int - разрешение изображения
2. 3 типа float - положение камеры в радиусах черной дыры (горизонта событий)
3. 3 типа float - направление камеры (главное направление, длина вектора не имеет значения)
4. 3 типа float - направление вектора up(куда смотрит вверх картинки, главное направление)
О массе и радиусе: масса дыры зафиксирована, радиус вычисляется по формуле Шварцшильда. 


Время работы программы: пару секунд (config1.txt); пару секунд (config2.txt); пару секунд (config3.txt);
Бонусы: 1. альфа смещивние
